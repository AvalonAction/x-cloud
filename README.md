# 开发注意
- 若配置文件出现乱码 请修改编译器文件编码格式为UTF-8
- 使用Lombok简化开发 开发时记得安装Lombok插件避免编译器报错
- 配置文件加密 Jasypt加密 可到xcloud-common服务中找到JasyptUtil工具类生成加密结果 使用到的模块配置文件中记得加上jasypt.encryptor.password
